Descoberta do desânimo de alunos em Ambientes Virtuais de Ensino e Aprendizagem, utilizando mineração de dados educacionais.

O Projeto



O projeto TECMIDES – Descoberta do Desânimo de Alunos em Ambientes Virtuais de Ensino e Aprendizagem Utilizando Mineração de Dados Educacionais tem como objetivo desenvolver um modelo para predizer alunos desanimados, com utilização da mineração de dados educacionais (MDE) e uma ferramenta no Moodle que, se acionada, obtém os alunos com propensão ao desânimo. Dessa forma, a possibilidade em instrumentalizar os cursos da UAB com uma ferramenta computacional, que identifique alunos desanimados, é importante na medida em que permite ao professor acompanhar melhor esse aluno, fazendo com que ele continue seu processo de aprendizagem evitando, inclusive, uma possível desistência.

Mineração de Dados

O projeto desenvolve um modelo para predizer alunos desanimados, com utilização da mineração de dados educacionais (MDE).


Plugin para o Moodle

Ainda em desenvolvimento, tem como objetivo fornecer ao professor uma ferramenta no Moodle que obtém os alunos com propensão ao desânimo.


Acompanhamento

A ferramenta possibilitará ao professor um melhor acompanhamento dos seus alunos em disciplinas EAD.
